# Abdul Hamid - Cyber Portfolio

A modern, tech-forward personal portfolio website for **Abdul Hamid**, an Independent Cybersecurity Researcher, ISP & Cloud Architect. Built with React 19, Tailwind CSS 4, and Framer Motion.

## 🚀 Features

- **Tech-Forward Aesthetic**: Dark mode by default with a "Cyber-Security" theme (Cyber Blue & Purple accents).
- **Glassmorphism UI**: Modern frosted-glass effect on cards and UI elements.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Interactive Elements**:
  - Light/Dark mode toggle.
  - Smooth scroll navigation.
  - Hover animations and staggered entrance effects.
  - Terminal-style "Whois" profile card.
- **Comprehensive Sections**:
  - **Hero**: Quick identity and system status.
  - **About**: Profile summary and technical identity.
  - **Skills**: Interactive grid of expertise.
  - **Experience**: Clean timeline of professional history.
  - **Certifications**: List of industry-recognized achievements.
  - **Contact**: Quick connection links (Email, LinkedIn, GitHub).

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 6](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/) (formerly Framer Motion)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Space Grotesk (Display), Inter (Sans), JetBrains Mono (Mono)

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ahamid/ahamid.github.io.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

### Build

Create a production build:
```bash
npm run build
```
The output will be in the `dist` directory.

## 🌐 Deployment to GitHub Pages

To deploy this project to GitHub Pages:

1. Ensure `base` in `vite.config.ts` matches your repository name (if not using a custom domain).
2. Build the project: `npm run build`.
3. Push the contents of the `dist` folder to the `gh-pages` branch or configure GitHub Actions for automatic deployment.
